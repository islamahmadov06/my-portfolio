# My Portfolio — İslam Əhmədov

**Repository:** `my-portfolio` (public)

**What this repo contains**
- `index.html` — Home page (about, education, volunteering, contact)
- `blog.html` — Blog & articles
- `projects.html` — Projects / Services / Gallery
- `css/style.css` — stylesheet
- `assets/` — profile photo, favicon and other media

## How to use
1. Replace placeholders in HTML files (GROUP_NUMBER, BIO, etc.) with your actual information.
2. Add your profile photo to `assets/profile.jpg` (or update the image path in the HTML).
3. Commit and push to a **public** repository on GitHub.
4. Publish on GitHub Pages: go to `Settings → Pages` and set source to `main` branch and `/ (root)`.
5. Your site will be available at `https://<your-github-username>.github.io/<repo-name>/`.

## Submission checklist
- [ ] Full name and group number included in site
- [ ] GitHub username added to README
- [ ] Repo is Public
- [ ] GitHub Pages published and link provided
